# A列車向けダイヤグラム生成

A列車の運用計画を作るための、路線図・運行経路・ダイヤグラム作成ツールです。

駅や車庫を配置して線路をつなぎ、区間ごとの所要時間と列車ごとの営業時間・回送時間を設定すると、停車時刻とダイヤグラムを自動計算できます。

- 作者: TRAPPIST-1E
- 更新日: 2026-05-01

## 特徴

- 駅、ターミナル駅、車庫、留置線、折返し、立体交差駅を配置できます。
- 接続点、番線、接続線、分岐器を使って路線図を編集できます。
- 待避型、片渡り、両渡りなどの分岐器を扱えます。
- 接続点間の区間に所要時間を設定できます。
- 同じ区間に複数の車速区分を持たせ、車速区分ごとに所要時間を切り替えられます。
- 分岐を含む区間では、車速区分ごとに分岐間の時間配分を設定できます。
- 営業経路と回送経路をまとめた経路セットを作成できます。
- 各列車に営業時間、回送時間、停車/通過設定、経路セット、車速区分を指定できます。
- 設定した所要時間から Stop 時刻とダイヤグラムを自動計算します。
- ライトテーマとダークテーマを切り替えられます。
- JSON 保存/読み込み、CSV 出力に対応しています。

## 使い方

1. Station に駅名を追加します。
2. 路線図・運行経路エディタで駅、車庫、留置線などのノードを配置します。
3. ノードの接続点同士を線でつなぎます。
4. 必要に応じて接続線上に分岐器を追加します。
5. 所要時間の設定モードで、対象区間と所要時間を設定します。必要に応じて車速区分を追加します。
6. 経路設定モードで、営業経路と必要な回送経路を作ります。
7. 列車運用で列車を追加し、営業時間、回送時間、停車設定、経路セット、車速区分を指定します。
8. 自動計算された Stop 時刻とダイヤグラムを確認します。
9. 必要に応じて JSON または CSV で出力します。

詳しい操作方法はアプリ下部の「操作方法」から確認できます。

## データの保存

編集内容はブラウザの `localStorage` に自動保存されます。

ただし、ブラウザのデータ削除、別端末での利用、公開環境の変更などでは失われる場合があります。重要なデータは `Download JSON` で保存してください。

保存した JSON は、アプリ下部の読み込み欄から復元できます。

サンプルとして使えるテンプレートデータは `template` に入っています。

## ローカル実行

```sh
npm install
npm run dev
```

5173 番で固定起動する場合:

```sh
npm run serve:local
```

ローカル確認用 URL:

```text
http://localhost:5173/
```

## ビルド

```sh
npm run build
```

公開用ファイルは `diagram-generation-tool/` に出力されます。

## Netlify で公開する場合

Netlify のトップページで公開する想定です。

```text
Build command: npm run build
Publish directory: diagram-generation-tool
```

`vite.config.ts` の `base` は `/` です。

サブディレクトリ配下で公開する場合は、`base` を公開パスに合わせて変更してください。

## プライバシー

このアプリは静的なブラウザアプリです。

- 入力した路線図や列車データは、通常は利用者のブラウザ内に保存されます。
- JSON/CSV を手動で出力しない限り、アプリ側が編集データを外部へ送信する処理はありません。
- API キーやパスワードなどの秘密情報を入力する用途は想定していません。

## 開発メモ

- フロントエンド: React / TypeScript / Vite
- スタイル: Tailwind CSS
- グラフ表示: Chart.js

主なファイル:

- `src/RouteNetworkEditor.tsx`: 路線図、接続線、分岐、所要時間、経路設定
- `src/TrainOperationSection.tsx`: 列車運用、経路セット、Stop 時刻
- `src/DiagramChartSection.tsx`: ダイヤグラム表示
- `src/StationMasterPanel.tsx`: 駅マスタ編集
- `src/ProjectFileSection.tsx`: JSON 読み込み、JSON/CSV 出力
- `src/lib/route-time.ts`: 分岐を含む区間所要時間の計算
