# ダイヤグラム生成ツール

A列車の運用計画向けに、路線図、運行経路、Stop時刻、ダイヤグラムをまとめて作成するブラウザアプリです。

駅や車庫などのノードを配置して線路をつなぎ、接続点間の所要時間、営業経路、回送経路、列車ごとの営業時間を設定すると、Stop時刻とダイヤグラムを自動計算できます。

公開ページ: [https://diagram-generation-tool.pages.dev/](https://diagram-generation-tool.pages.dev/)

- 作者: [TRAPPIST-1E](https://github.com/mm-trappist-1e/diagram-generation-tool)
- 更新日: 2026-05-08

## 主な機能

- 複数ワークスペースをタブ形式で管理できます。
- ワークスペースごとにJSON保存、JSON読込、CSV出力ができます。
- 駅、車庫、留置線、折返し、立体交差ノードを配置できます。
- 各ノードには番線数、番線名、終端設定を持たせられます。
- ノードは90度単位の回転と、接続点・番線方向の裏返しに対応しています。
- 接続点同士を接続線でつなぎ、線路形状を作成できます。
- 待避型、片渡り、両渡りなどの分岐器を追加できます。
- 接続線ごとに手動中継点を設定し、自動ルートを補正できます。
- 接続点間の所要時間区間を設定できます。
- 分岐を含む所要時間区間では、分岐間の時間配分を調整できます。
- 所要時間には複数の車速区分を持たせられます。
- 車速1を基準に、倍率で他の車速区分を作ることもできます。
- 営業経路と回送経路をまとめた経路セットを作成できます。
- 経路設定は下書きとして編集し、保存した時点で経路セットへ反映されます。
- 列車ごとに経路セット、車速区分、営業時間、回送時間、停車/通過を設定できます。
- 標準停車分は5分固定です。
- 所要時間と停車/通過設定からStop時刻を自動計算します。
- 作成した列車設定からダイヤグラムを自動描画します。
- ライトテーマ、グレーテーマ、ダークテーマを切り替えられます。

## 基本的な流れ

1. 必要に応じてワークスペースを追加、切り替え、複製します。
2. ノード追加で、路線図に使うノード名を登録します。
3. 駅、車庫、留置線などのノードをキャンバスへ配置します。
4. ノードの番線数や終端設定を調整します。
5. 接続点同士をつないで線路を作ります。
6. 必要に応じて接続線へ分岐器を挿入します。
7. 所要時間で、接続点間の対象経路と所要時間を設定します。
8. 車速区分が必要な場合は、区分ごとの所要時間を設定します。
9. 経路設定で、営業経路と必要な回送経路を作ります。
10. 列車を追加し、経路セット、車速区分、営業時間、回送時間、停車/通過を設定します。
11. 自動計算されたStop時刻とダイヤグラムを確認します。
12. 必要に応じてJSONまたはCSVで出力します。

## キャンバス操作

- 左端のアイコン列から、ノード追加、ルート修正、経路設定、所要時間、車速区分を開きます。
- 選択中のアイコンをもう一度押すと、そのツールを閉じます。
- ツール選択中にキャンバス上で右クリックすると、ツールと設定モードを閉じます。
- 空白ドラッグでキャンバスを移動します。
- ホイールまたはピンチ操作で拡大縮小します。
- 空白クリックで選択を解除します。
- Shift + 左ドラッグで範囲選択できます。
- Delete / Backspace で選択中の対象を削除できます。
- Ctrl + C / Ctrl + V で選択ノードをコピー、貼り付けできます。
- Ctrl + Z / Ctrl + Y で戻す、やり直しができます。

## ノードと接続線

- ノード追加では、ノード名、種類、番線数を指定してノードを追加します。
- 新規ノードは、現在表示しているキャンバス範囲の中央付近に追加されます。
- ノードはドラッグで移動できます。
- ノード付近の回転ボタンで90度回転できます。
- 裏返しボタンで、接続点と番線の向きを反転できます。
- 終端を有効にすると、片側だけ接続点を出せます。
- Ctrl + 接続点ドラッグで接続線を作成します。
- 未接続の接続点は白抜きで表示されます。
- 接続線をクリックすると選択できます。
- Ctrl + 右クリックで、選択対象や接続線を削除できます。

## 分岐器

- Shift + 左クリックで、分岐器を入れる接続線を選択します。
- Ctrl + Shift + 左ドラッグで、選択した接続線へ分岐器を挿入します。
- Ctrl + Shift + 左クリックで、単独の分岐器を追加できます。
- 1本の接続線を選択している場合は、ドラッグ方向で待避分岐の開く向きを決めます。
- 2本の接続線を選択している場合は、接続線側に応じて片渡りZ/逆Zを切り替えます。
- 分岐器を選択すると、ノード編集で形状や番線を変更できます。

## ルート修正

- ルート修正アイコンを押した状態で接続線をクリックすると、修正を開始します。
- クリックで中継点を追加します。
- Shift + クリックで直交補助つきの中継点を追加します。
- Ctrl + Z / Ctrl + Y で中継点の追加を戻す、やり直しできます。
- 保存を押すと、中継点が接続線へ反映されます。
- 自動に戻すを押すと、手動中継点を削除して自動ルートへ戻します。
- 右クリックまたはEscで中断します。
- ノードを移動した場合、手動中継点は自動ルートへフォールバックします。

## 所要時間と車速区分

- 所要時間の設定モードでは、開始接続点、必要な分岐、終端接続点の順に選択します。
- 対象経路が完成したら所要時間を入力して保存します。
- 設定済み区間をクリックすると、所要時間や分岐ごとの時間配分を編集できます。
- 設定済み区間を右クリックすると、進行方向を順、逆、双方向に切り替えられます。
- 順と逆は表示上の始点/終点を入れ替えて矢印で表現します。
- 双方向は双方向矢印で表示します。
- 車速区分を追加すると、同じ区間に別の所要時間セットを持たせられます。
- 倍率を使う場合は、車速1を基準に各区分の所要時間を一括変換します。

## 経路セット

- 経路設定では、営業経路と回送経路を持つ経路セットを作成します。
- 基本は営業経路を設定します。
- 回送経路を使う場合は、経路セット側で回送経路を有効にします。
- 設定モード中は、キャンバス上のノードの番線領域を順番に選択します。
- 設定モード中のクリック操作は、すぐには経路セット本体へ反映されません。
- 下の一覧には最新の下書きが表示されるため、現在設定中の経路を確認できます。
- 保存を押すと、営業/回送の下書きが経路セットへ反映されます。
- 戻す、クリアは下書きだけを変更します。
- 設定モードを閉じたり別ツールへ切り替えたりした場合、未保存の下書きは破棄されます。

## 列車運用とダイヤグラム

- 列車一覧から列車を追加できます。
- 列車は複製、削除、ドラッグ並び替えに対応しています。
- 列車ごとに経路セット、車速区分、営業時間、回送時間を設定します。
- 表の状態欄で停車/通過を変更できます。
- 回送時間中は回送経路を走行します。
- 営業終了後や回送終了時刻の扱いは、経路セットと停車/通過設定に基づいて計算されます。
- Stop時刻は所要時間と停車/通過から自動計算されます。
- ダイヤグラムの読取方向は、ダイヤグラム欄で指定できます。

## 保存と出力

編集内容はワークスペース単位でブラウザ内に自動保存されます。

ただし、ブラウザのデータ削除、別端末での利用、シークレットモード、公開URLの変更などでは保存内容が失われたり参照できなくなったりする場合があります。重要なデータは、ダウンロードアイコンからJSONとして保存してください。

- JSON保存: 現在のワークスペースを後から復元できる形式で保存します。
- JSON読込: 保存したJSONを現在のワークスペースへ読み込みます。
- CSV出力: 外部利用向けに時刻データを出力します。編集データの完全保存にはJSONを使ってください。

初回起動時のワークスペース1にはテンプレートデータが読み込まれます。新規ワークスペースは空の状態で開始します。

## プライバシー

このアプリはブラウザ上で動作する静的なアプリです。

- 入力した路線図や列車データは、通常は利用者のブラウザ内に保存されます。
- JSONやCSVを手動で出力しない限り、編集データをアプリ側が外部へ送信する処理はありません。
- APIキー、パスワード、個人情報などの秘密情報を入力する用途は想定していません。
